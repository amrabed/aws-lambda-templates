## 2025-05-14 - [Documentation Accessibility: Alt Text]
**Learning:** In backend-heavy projects or template repositories, the documentation (e.g., MkDocs) is the primary user interface. Architecture diagrams often lack descriptive alt text, which is a critical accessibility barrier for screen reader users.
**Action:** Always check documentation for images with empty or missing alt text (`![]()`) and provide concise, descriptive summaries of the visual information.
